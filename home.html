Alguns Projetos

Scraping do Site BooksToScrap

Este projeto utilizou Python e Selenium para realizar web scraping no site BooksToScrap. Os dados extraídos, como título, gênero e preço, foram armazenados em listas e, posteriormente, convertidos para um DataFrame do Pandas. O símbolo de euro foi removido dos preços e os valores foram convertidos para o tipo float, permitindo análises numéricas. Os dados finais foram exportados para um arquivo CSV. Confira funcionando aqui.
